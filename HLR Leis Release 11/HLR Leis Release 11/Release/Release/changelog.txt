Квары и их описание ( мин/макс - описание ) :

[Aimbot]

1) aim_Active - ( 0/1 - выключает/включает аим )
2) aim_Deathmatch - ( 0/1 - выключает/включает работу аима по вашей команде )
3) aim_Aimthrough - ( 0/1 - выключает/включает работу аима через стены )
4) aim_Silent - ( 0/2 - (0 - выключает) (1 - включает невидимую наводку для вас но видемую для наблюдателей)
( 2 - частичный perfect silent иногда при 1 выстрее вас могут не увидеть наблюдатели как вы убили с аима, шансы 40 что сработает и 60 что не сработает )
5) aim_Smooth - ( 0/10 - сглаживание наведения аима )
6) aim_Fov - ( 1/180 - угол срабатывания аима )
7) aim_FovType - ( 0/2 - (0 - обычный статичный fov на всех расстояниях один)
 (1 - чем дальше игрок тем больше fov, за минимальную дистанцию для увеличения fov - квар aim_FovDist )
 (2 - чем дальше игрок тем меньше fov за максимальную дистанция для уменьшения fov - квар aim_FovDist ) )
8) aim_FovDist ( 5/7000 - нужен для настройки размерности aim_Fov при aim_FovType 1/2 )
9) aim_NoTwitching ( 0/1 - выключает/включает аим, когда был убил противник и в fov попал кто то другой )
10) aim_Mode - ( 0/2 - origin,bone,hitbox тут все понятно )
11) aim_Recoil - ( 0/100 - опускание прицела при стрельбе )
12) aim_Origin - ( 0/22 - точка наведение аима при aim_Mode 0 )
13) aim_Bone - ( 0/22 - точка наведение аима при aim_Mode 1 )
14) aim_Hitbox - ( 0/22 - точка наведение аима при aim_Mode 2 )
15) aim_Time - ( 0/5000 - время работы аима при попаданиие прицела в fov ) время в ms
16) aim_Delay - ( 0/3000 - задержка перед наведением аима ) время в ms
17) aim_ShotDelay - ( 0/500 - задержка перед первым выстрелом аима для убийства цели с первой пули ) время в ms
18) aim_KillDelay - ( 0/10000 - отключает аим на заданное время в ms после убийста противника )
19) aim_Prediction - ( -1000/1000 - предсказание для наведения аима )
20) aim_Distance - ( 0/1 - (0 - целится на игрока который ближе к прицелу)
 (1 - целиться на игрока который ближе по расстоянию) посмотреть можете принцип через aim_DrawFov )
21) aim_DrawAimspot - ( 0/1 - выключает/включает рисование точки куда будет целится аим ) (только в глобальной настройки Aimbot в ini)
22) aim_DrawFov - ( 0/1 - выключает/включает рисование fov возле точки (aim_DrawAimspot) ) (только в глобальной настройки Aimbot в ini)
23) aim_Crosshair - ( 0/1 - выключает/включает рисование точки только для awp ) (только в глобальной настройки awp в ini)
24) В меню aim_SaveSettings - сохраняет настройки для текущего оружия в файл
25) В меню aim_LoadSettings - загружает настройки для текущего оружия из файла

[Knifebot]

1) knf_Active - ( 0/1 - выключает/включает авто удар на ноже )
2) knf_Aim - ( 0/1 - выключает/включает наводку с любого угла knf_AimFov )
3) knf_AimFov ( 90/360 - угол срабатывание при knf_Aim )
4) knf_Team - ( 0/1 - выключает/включает атаку по своей команде )
5) knf_Attack - ( 1/2 - тип удара mouse1/mouse2 )
6) knf_DistAttack - ( 1/100 - минимальная дистанция для удара mouse1 при knf_Attack 1 )
7) knf_DistAttack2 - ( 1/100 - минимальная дистанция для удара mouse2 при knf_Attack 2 )

[Esp]
1) esp_DrawFont ( 0 - отрисовка шрифта через движок кс , 1 - через opengl )
2) esp_MultiColor ( 0 - цвета отрисовки будут зелёные если данный объект виден вам и красный если не виден, 1 - мульти цвета (для esp_weapon,sequence,gaitsquence) будут разные )

[Radar]
rad_Type - ( 0/2 - (0 - показывает игроков) (1 показывает звуки) (показывает игроков + звуки) )

[Removal]
rem_FlashMessage - ( 0/1 - выключает/включает отрисовку сообщения о noflah )
rem_AmxxMessage - ( 0/1 - выключает/включает отрисовку сообщений amxx от сервера )

[Misc]
misc_FreeLook - ( 0/1 - выключает/включает свободный полёт по карте, например для поисках секретных помещений в ней )
misc_FreeLookSpeed - ( 100/2000 - скорость полёта в misc_FreeLook )
misc_AntiScreen - ( 0/1 - выключает/включает все визуалы чита при консольных команд snapshot,screenshot )
misc_GranadeTrail - ( 0/1 - выключает/включает показ куда полетит граната если нажмёте mouse2 взяв её )
misc_FastRun - ( 0/1 - быстрый бег с ножом в руках )
misc_MenuColor - ( 0/1 - выключает/включает подцветку кваров в меню )
